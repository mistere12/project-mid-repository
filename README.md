# Project Description

This project is a simple Java console program made using Object-Oriented Programming concepts.

The system is used to manage animals in an animal shelter. The user can:

* add animals,
* adopt animals,
* and display available animals.

The program contains three animal types:

* Dog
* Cat
* Bird

Each animal has:

* ID
* Name
* Age
* Gender

The project was developed using the concepts learned from the OOP course materials.

---

# System Features

## Add Animal

The user can:

* choose animal type,
* enter animal information,
* and store the animal in the system.

## Adopt Animal

The user can:

* choose animal type,
* enter the animal ID,
* and adopt the animal.

## Display Animals

The user can:

* choose animal type,
* and display available animals of that type.

# OOP Concepts Used

## Classes and Objects

The project uses different classes such as:

* Animal
* Dog
* Cat
* Bird
* Main

Objects are created from these classes.

---

## Encapsulation

Private variables are used in the Animal class to protect data.

Getter and setter methods are used to access and modify the data.

---

## Inheritance

Dog, Cat, and Bird classes inherit from the Animal class using the `extends` keyword.

This helps reuse code from the parent class.

---

## Polymorphism

### Method Overriding

The child classes override the `displayInfo()` method from the Animal class.

### Runtime Polymorphism

The program uses an Animal array to store Dog, Cat, and Bird objects together.

---

## Method Overloading

The `show()` method in the Animal class is overloaded using different parameters.
